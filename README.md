# Classification models study

This repository contains the study of several classification models, as well as the development of the naive-bayes classification algorithm in python of the subject "Intelligent Systems"  from the Master in Computer Engineering of the University of Salamanca.
The following dataset has been used for the tests: https://www.kaggle.com/iamsumat/spotify-top-2000s-mega-dataset
